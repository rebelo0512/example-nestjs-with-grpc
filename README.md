## Description

Um exemplo de implementação de um servidor e um cliente GRPC no framework NestJS.
Para testar basta iniciar ambos os serviços com 'yarn start:dev ou npm run start:dev'.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
